Introduction to CSS

CSS stands for Cascading Style Sheets. It's used to style the layout and appearance of web pages. CSS controls the color, font, spacing, and positioning of HTML elements.

Basic syntax: A CSS rule consists of a selector and a declaration block. The syntax looks like this:

      selector {
        property: value;
      }

Example:

      h1 {
        color: blue;
      }

CSS Examples

Example 1: Change Text Color

In the example below, we will change the color of the text using the CSS `color` property.

        p {
          color: red;
        }

Result:

This is a red colored text.

Example 2: Changing Background Color

Here, we change the background color of an element using the `background-color` property.

        div {
          background-color: lightblue;
        }
This div has a light blue background.
