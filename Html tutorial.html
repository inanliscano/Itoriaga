1. Basic Structure of an HTML Page

An HTML document begins with <!DOCTYPE html> to define the document type and version. The structure of the page includes the <html>, <head>, and <body> tags.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Website</title>
</head>
<body>
  <h1>Welcome to My First Website</h1>
  <p>This is a basic HTML tutorial.</p>
</body>
</html>
